# wordle-clone

Six tries to guess the 5-letter word.

**Green** = right letter, right spot. **Yellow** = right letter, wrong spot. **Gray** = not in the word.

## Features

- Daily mode — same word for everyone, resets at midnight
- Free play mode — new word whenever you want
- On-screen keyboard with color-coded feedback
- Tile flip animation, shake on invalid word
- Win/loss stats and streak counter

## Run

```bash
npm install && npm run dev
```

Word list is the original Wordle list (2315 common English words).
